# Căutarea datelor

## Introducere

geo-spatial.org oferă două mecanisme de căutare/identificare a datelor:
- [Catalogul interactiv](https://services.geo-spatial.org/geonetwork/srv/eng/catalog.search#/search) bazat pe GeoNetwork;
- [Catalogul static](/descarcare) de explorare a datelor.

:::info[De ce este nevoie de ambele mecanisme?]

Pe scurt, pentru a oferi atât funcționalități avanasate cât și o manieră simplă și robustă de explorarea datelor. GeoNetwork oferă o interfață ergonomică, un motor de căutare state-of-the-art (bazat pe [Elasticsearch](https://www.elastic.co/elasticsearch/)), suport pentru volume mari de date și integrarea directă a datelor în hărți interactive. Funcționalitatea avansată poate duce uneori și la probleme de natură tehnică datorită numărului mare de dependențe și complexitatea arhitecturii. Pe de altă parte, catalogul static oferă o manieră simplă și intuitivă de explorare a datelor, cu acces direct la resursele asociate ce depinde doar de funcționarea unui server HTTP, neavând componente dinamice.

:::

## Catalogul interactiv GeoNetwork

Catalogul interactiv este un catalog de date bazat pe aplicația open-source [GeoNetwork](https://geonetwork-opensource.org). Aceasta oferă o interfață intuitivă pentru descoperirea rapidă a seturilor de date, serviciilor și aplicațiilor disponibile în catalogul geo-spatial.org. Utilizatorul poate realiza căutări simple sau avansate, folosind cuvinte-cheie, filtre tematice sau geografice și poate accesa metadatele și resursele asociate direct din interfață.

### Accesarea interfeței de căutare

După [accesarea platformei GeoNetwork](https://services.geo-spatial.org/geonetwork/srv/eng/catalog.search#/search), utilizatorul va fi întâmpinat cu o pagină de start care include o bară de căutare centrală, o listă cu seturile de date indexate în zona centrală și un meniu de filtre în partea stângă.

### Căutare simplă (full-text)

Pentru a efectua o căutare rapidă:

- Se introduce un cuvânt-cheie sau o expresie (ex: „județe”, „hidrografie”, „UAT”) în bara principală de căutare.
- Sistemul returnează toate metadatele care conțin termenul căutat în titlu, descriere, cuvinte-cheie sau alte câmpuri relevante.
- Rezultatele pot fi sortate după **relevanță**, **ultima actualizare**, **data publicării**, **titlu** **rating** și **popularitate**.

---

### Căutare avansată

Pentru rezultate mai precise, utilizatorul poate utiliza filtrele din partea stângă a interfeței. Criteriile disponibile includ:

- **Tip resursă**: set de date, serviciu, document, etc.
- **Organizații**: permite filtrarea după organizația implicată în producția/distribuția datelor.
- **Tip de date**: vector, raster, etc.
- **Tipul de serviciu prin care datele sunt accesibile**: vizualizare, descărcare, API, etc.
- **Formatul fișierului**: GeoPackage, Shapefile, COG, etc.
- **Scara de referință**

De asemenea, prin utilizarea butonului <i className="fa-solid fa-sliders"></i> (Search options) din dreapta barei de căutare, se poate limita intervalul de timp când datele au fost publicate sau revizuite.

---

### Căutare geografică (spațială)

GeoNetwork permite restrângerea rezultatelor pe baza unei **extinderi geografice**:

- Utilizatorul poate folosi harta interactivă din colțul din stânga-jos și să traseze un **dreptunghi de selecție** pentru a limita căutarea la o zonă de interes (date ce intersectează zona indicată sau care se află complet în interiorul zonei indicate).

---

### Vizualizarea și descărcarea rezultatelor

După identificarea unui set de date relevant, utilizatorul poate:

- Accesa fișa completă de metadate.
- Consulta informații detaliate despre sursă, rezoluție/scară, licență, acoperire geografică și temporală.
- Descărca datele sau accesa serviciile web (WMS, WFS etc.) folosind linkurile din secțiunea „Resurse online”.
- Previzualiza datele folosind harta interactivă inclusă direct în fișa de metadate.

## Catalogul static

Catalogul static este un catalog de date bazat pe descărcarea datelor. Aceasta este o pagină web care permite descărcarea datelor disponibile.
