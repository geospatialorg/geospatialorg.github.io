# Adăugarea unui nou set de metadate în GeoNetwork

Publicarea unui nou set de date în catalog începe cu descrierea sa printr-un fișier de metadate conforme cu standardul ISO 19115 / 19139. GeoNetwork ne oferă o interfață web intuitivă pentru acest proces.

## Autentificarea în GeoNetwork

Accesați [interfața web a catalogului GeoNetwork](https://services.geo-spatial.org/geonetwork) și autentificați-vă cu contul dumneavoastră folosind butonul din dreapta-sus. Dacă nu aveți cont, consultați secțiunea Despre.

## Crearea un nou fișier de metadate

Crearea unei noi intrări în catalog se face accesând meniul `Contribute/Add new record`.

<figure style={{ textAlign: "center" }}>
  <img src={useBaseUrl('/img/ghiduri/metadate/geonetwork-add-new-record.jpg')} alt="Crearea un nou fișier de metadate" />
  <figcaption><em>Crearea un nou fișier de metadate.</em></figcaption>
</figure>

În fereastra următoare, se vor alege:
* Tipul de resursă pentru care se vor adăuga metadate (cel mai probabil, `Dataset`) din secțiunea `Create a Dataset`;
* Șablonul corespunzător tipului de date pentru care se vor adăuga metadatele (`Template for Vector data in ISO19139 (preferred!)` pentru date vectoriale sau `Template for Raster data in ISO19139` pentru date raster) din secțiunea `From`;
* Grupul căruia îi va aparține fișierul de metadate.

:::info

În GeoNetwork, **grupurile** sunt mecanisme de organizare a utilizatorilor și metadatelor, care permit gestionarea colaborativă a conținutului în cadrul unei comunități. În cazul catalogului geo-spatial.org, fiecare grup reprezintă o temă (ex. „Limite administrative”). Atunci când creați sau editați un set de metadate, trebuie să alegeți grupul relevant.

:::
